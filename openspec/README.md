# OpenSpec

This directory holds the spec-driven development (SDD) artifacts for this project.

- `changes/` — in-flight change proposals, each with its own proposal, specs, design, and tasks.
- `specs/` — the archived, authoritative capability specs (populated when changes are archived).

The authoritative source of truth for in-flight artifacts is Engram
(`sdd/{change-name}/{artifact}`); the files here are materialized mirrors for
review and offline reference.
